Simple TCP Client - Server code in Python:
Based on one client updating two servers at 5 seconds intervals.
-----------------------------

List of Files:
--------------
server1/server1_coding_test_1_1.py
server2/server2_coding_test_1_2.py
client/tcp-client.py

How to Run:
1. Update HOST1_NAME, HOST2_NAME in client/tcp-client.py to point to the correct servers
2. Run the servers using
      python server1_coding_test_1_1.py
      python server2_coding_test_1_2.py
3. Run the client
      python tcp-client.py
4. Observe the log files under server1, server2 directory
